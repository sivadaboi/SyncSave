    How It Works

    1
    Generate or share a code

    Create a unique room code and share it with the other device via any messenger.

    2
    Both devices join the room

    Enter the same code on both machines. The SaveSync cloud relay passes traffic between room members — no port forwarding needed.

    3
    Pair & sync automatically

    Once in the same room, SaveSync handles pairing and keeps your saves in sync — even across continents.

    4
    Want to self-host?

    Deploy the relay to Render.com for free using the render.yaml in the project root, or use Docker.

    About the .sscb Format

    🗜️
    Brotli Compression

    Industry-leading lossless compression (up to 90%+ reduction on text-heavy save data).

    📁
    Per-Game Files

    Each game's saves are packed into its own .sscb file so you can restore one game without extracting everything.

    📋
    Metadata Manifest

    Every backup folder contains a backup-metadata.json with original paths, timestamps, and compression statistics.
